## New Business

### Item 1: Warmup 2 Project Status
- **Introduced by:** Evan Marriott
- **Discussion:** Team has completed repo setup, conducted research over the weekend, and started the iteration process. Theme is a "broke college student" slot machine with icons like ramen noodles. Features planned include auditory feedback (coin sounds on win) and color psychology informed by research on slot machine engagement. Four iteration stages are defined: (1) foundation/architecture, (2) math and logic, (3) connect UI and logic, (4) bug fixes and hardening. Each stage has two people assigned, with a target completion of the following afternoon.
- **Resolution / Status:** Resolved — TA approved the plan

### Item 2: Repo Organization
- **Introduced by:** Evan Marriott
- **Discussion:** Team proposed a single `game/` folder with subfolders for each iteration step, where each pair builds on the previous pair's output. TA confirmed this is acceptable as long as the structure is logical. TA also recommended including a folder structure diagram in the README.
- **Resolution / Status:** Resolved — current structure approved; README diagram already in place

### Item 3: Warmup 2 Prompt Guidelines
- **Introduced by:** Audria (TA)
- **Discussion:** Minimum of 20 prompts required; no maximum length restriction. TA recommended making prompts longer and more detailed to get better output from the LLM. Grading is based on the research phase, the final product, and whether implemented features align with the user studies conducted.
- **Resolution / Status:** Resolved — guidelines clarified
