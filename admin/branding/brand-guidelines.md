*Accessibility Note: Always ensure text has a high contrast ratio against its background. Use white (`#FFFFFF`) or very light gray text on Prussian Blue and Twilight Indigo backgrounds.*

## 🔤 Typography

We use two primary typefaces to balance a clean, modern UI with a technical, developer-focused aesthetic. Both fonts are freely available via Google Fonts.

### 1. Inter (Primary UI & Headings)
**Inter** is a highly legible, modern sans-serif designed specifically for computer screens. 
* **Usage:** Headers (`h1` - `h6`), body text, buttons, navigation, and general UI elements.
* **Weights:** * *Regular (400)* for standard body text.
  * *Medium (500)* for buttons and subtitles.
  * *Bold (700)* for main headers and emphasis.
* **CSS Fallback:** `font-family: 'Inter', system-ui, -apple-system, sans-serif;`

### 2. Fira Code (Monospace & Technical)
**Fira Code** is a monospace font featuring programming ligatures, giving our technical specs a premium feel.
* **Usage:** Code snippets, IDE-like interfaces, terminal outputs, technical documentation blocks, and API keys.
* **Weights:** * *Regular (400)* for standard code blocks.
  * *Medium (500)* for emphasized variables.
* **CSS Fallback:** `font-family: 'Fira Code', 'Courier New', Courier, monospace;`
