### Item 3: Slot Machine Feature Planning
- **Introduced by:** Nicole Sutedja, Evan Marriott
- **Discussion:** Team shared ideas for features informed by research on slot machine engagement. Key suggestions included:
  - Auditory feedback — coin sounds on wins, unique sounds per interaction (Jensen Guo, Aron Wu)
  - Visual animations and special effects on payouts (Benedict Luis)
  - Image-based symbols instead of plain emojis/text (Han Yang-Lin)
  - Color psychology — warm colors for excitement, cool colors for calm (Kaley Chung)
  - Real money simulation mode vs. in-game tokens for fun (Evan Marriott)
  - Mini-games tied to wins (Bethany Miyamoto)
  - Small bet sizing to fit the broke college student theme (Evan Marriott)
- **Resolution / Status:** Resolved — features documented; to be referenced when prompting the AI

### Item 4: Theme Vote
- **Introduced by:** Nicole Sutedja, Bethany Miyamoto

### Item 6: Repo Structure — Game Folder and Iterations
- **Introduced by:** Evan Marriott

This allows the team to show clear progression between phases in the final presentation. No branches needed for now since teams are working in isolated folders.
- **Resolution / Status:** Resolved — `game/` folder with four subfolders created and pushed by Evan

### Decision 2: Four-folder iteration structure inside `game/`

- **Reasoning:** Keeps phases clearly separated, enables progress to be shown during the final presentation, and keeps the linter path-stable

- **Why the group proceeded anyway:** N/A

- **Reasoning:** Low collision risk given folder isolation; branching adds overhead for a short-turnaround project
